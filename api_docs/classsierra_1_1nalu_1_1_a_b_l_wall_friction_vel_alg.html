Detailed Description

template<typename BcAlgTraits>
class sierra::nalu::ABLWallFrictionVelAlg< BcAlgTraits >

Compute the wall friction velocity at integration points for the wall boundary of a given topology.

In addition to computing the friction velocity (utau) at the integration points, it also computes a partial sum (utau * area) and (area) at the integration points that is used to compute the area-weighted average utau over the ABL wall by WallFricVelAlgDriver.

See also
WallFricVelAlgDriver, BdyLayerStatistics
