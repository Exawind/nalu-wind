Detailed Description

Options to control TIOGA overset hole-cut behavior.

This class is a simple structure that holds the different options that can be used to control the behavior of TIOGA during holecuts. These options can be set from the Nalu-Wind input file within the overset boundary condition section and are passed to TIOGA during runtime.

◆ nFringe_

int tioga_nalu::TiogaOptions::nFringe_ {1}
private

Number of fringe layers on the overset BC sideset.

This parameter indicates the number of layers that will be set as mandatory fringes on an overset boundary. This default for this parameter is set in TIOGA API, so an additional flag is used to determine if the user has provided this variable in the input file.
