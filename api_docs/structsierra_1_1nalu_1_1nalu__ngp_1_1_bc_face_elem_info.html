Detailed Description

template<typename Mesh = stk::mesh::NgpMesh>
struct sierra::nalu::nalu_ngp::BcFaceElemInfo< Mesh >

Lightweight data structure for face/elem pair information in mesh loops.

The bucket loop wrappers provide this object to the lambda function. The connected nodes are always for the parent element that owns the boundary face.
