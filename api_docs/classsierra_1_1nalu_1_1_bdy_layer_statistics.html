Detailed Description

Boundary layer statistics post-processing utility.

A post-processing utility to compute statistics of flow quantities that are temporally and spatially averaged. Primarily used to evaluate atmospheric boundary layer characteristics during precursor simulations, this utility can also be used for channel flows.

The temporal averaging is perfomed via sierra::nalu::TurbulenceAveragingPostProcessing class.
