Detailed Description

template<typename AlgTraits>
class sierra::nalu::GeometryInteriorAlg< AlgTraits >

Compute nodal/element volumes and edge area vectors.

"edge_area_vector" is only computed if the user has requested edge-based finite-volume in the input file.

See also
GeometryAlgDriver, GeometryBoundaryAlg
