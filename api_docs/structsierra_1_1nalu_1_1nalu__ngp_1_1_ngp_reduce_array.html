Detailed Description

template<typename ScalarType, int N>
struct sierra::nalu::nalu_ngp::NgpReduceArray< ScalarType, N >

A custom Kokkos reduction operator for array types.

Useful when you want to accumulate multiple quantities, e.g., computing an area-weighted average.
