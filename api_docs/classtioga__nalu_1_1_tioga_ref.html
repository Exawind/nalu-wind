Detailed Description

Manager for the TIOGA handle.

This class manages access to the global TIOGA instance across multiple realms as well as across multi-solver interfaces (e.g., amr-wind/nalu-wind coupling). This class allows both use-cases: 1. nalu-wind only run where overset might be coupling within just one realm or multiple realms but owns the TIOGA reference, 2. hybrid solver run where an external code interface owns the TIOGA reference.
