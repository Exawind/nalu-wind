Detailed Description

template<typename Mesh = stk::mesh::NgpMesh, typename FieldManager = nalu_ngp::FieldManager>
class sierra::nalu::nalu_ngp::MeshInfo< Mesh, FieldManager >

STK mesh object holder.

This lightweight class carries information regarding the STK meshes both the non-NGP versions (MetaData/BulkData) as well as the stk::mesh::NgpMesh and nalu_ngp::FieldManager instances.
