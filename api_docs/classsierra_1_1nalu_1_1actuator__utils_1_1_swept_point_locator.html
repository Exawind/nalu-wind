Detailed Description

Implementation of a periodic Bezier curve (Sanchez-Reyes, 2009) to connect points at a specific radius The advantage of this method is it maps distorted points to an elipsoide with fewer samples than pure B-Splines or Bezier curves.

Fewer points are needed to create a perfect circle in the case of equispaced points (min =3) It is a parametric curve over the interval [0, 2pi]
